You can use Engi online on our server but running it locally on your own computer has some benefits. That way you can use your own textures, videos, 3D scenes and audio files, and you can publish the html straight to your hard drive. While there isn't an installer file for Engi yet, these instructions should get you up and running in no time.


INSTALL FOR OSX + LINUX

1. Install Git
2. Install Node.js (On the mac you can use Homebrew)
3. Run this in terminal:
$ git clone https://github.com/engijs/engi.git
$ cd engi
$ npm install
$ node server
4. Point your browser to http://localhost:8000/.


INSTALL FOR WINDOWS

1. Install Express version of Visual C++. This is needed to build native Node module extensions.
2. Install Git
3. Install Node.js
4. Run this in a command prompt window
$ git clone https://github.com/engijs/engi.git
$ cd engi
$ npm install
$ node server
5. Point your browser to http://localhost:8000/.
