USE IT ON YOUR OWN COMPUTER

You can use Engi online on our server but running it locally on your own computer has some benefits. That way you can use your own textures, videos, 3D scenes and audio files, and you can publish the html straight to your hard drive. See our installation page for more details.


SOURCE CODE

PRESETS

If you are using engi on your local computer, you can add your own presets by editing /presets/presets.json file.


USING YOUR OWN 3D SCENES

Scenes are converted via Blender (minimum required version is 2.69). Download the latest blender version and install our converter plugin by copying or linking /tools/blender/io_engi_export.py to your [blender]/[version]/scripts/addons path. Remeber to enable the plugin in your user preferences.


CONVERTING VIDEO FILES

Videos are .ogv files. Converting works with ffmpeg using these arguments:
ffmpeg -i inputfile.mpg -an -vcodec libtheora -vb 1024k -s 256x256 -aspect 1:1 output.ogv
