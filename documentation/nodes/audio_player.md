# Audio player

## Description
Play an audio stream. Playback loops.

## Inputs
### audio

*Audio*

The audio stream to play.

### play

*Boolean*

Send true to start playback and false to stop.

### mute

*Boolean*

Send true to mute playback and false to reenable audio.

### volume

*Float*

Set playback volume.

### time

*Float*

Set playback time.

## Outputs
n/a
