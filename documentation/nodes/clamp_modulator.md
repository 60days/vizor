# Clamp

## Description
Emit a float **value** no less than **min** and no greater than **max**.

## Inputs
### value

*Float*

Value to be clipped.

### min

*Float*

Minimum output value.

### max

*Float*

Maximum output value.

## Outputs
### result

*Float*

Emits **min** <= **value** >= **max**.
