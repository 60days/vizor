# Blend color

## Description
Color x-fader. Perform linear blend between two colors.

## Inputs
### color A

*Color*

First color operand.

### color B

*Color*

Second color operand.

### mix

*Float*

0: Emit pure color A 1: Emit pure color B

## Outputs
### color

*Color*

Linear mix of color A and B: A * (1 - mix) + B * mix
