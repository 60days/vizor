# Stereo Cubemap

## Description
Load a stereo cubemap from a URL. JPEG and PNG supported. Hover over the Browse button to select an existing image from the library.

## Outputs
### left

*CubeTexture*

The left side of the loaded stereo cubemap.

### right

*CubeTexture*
