# Video player

## Description
Play a video stream. Playback loops by default.

## Inputs
### video

*Video*

The video stream to play.

### play

*Boolean*

Send true to start playback and false to stop.

### muted

*Boolean*

Is audio muted?

### volume

*Float*

Set playback volume.

### time

*Float*

Set playback time.

## Outputs
### texture

*Texture*

The current video frame.
